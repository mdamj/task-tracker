TASK TRACKER APP:

A full-stack Task Management Application built using Next.js and Supabase, featuring secure authentication and complete task CRUD functionality. The app is deployed on Vercel.

TECH STACK:

Frontend: Next.js, React, Tailwind CSS

Backend & Auth: Supabase (PostgreSQL + Auth)

Deployment: Vercel

Language: JavaScript

FEATURES:

User authentication (Sign up / Login / Logout)

Create, read, update, and delete tasks

Task filtering and status management

Secure user-specific data using Supabase Auth

Responsive UI for all screen sizes

Production deployment with environment variables

PROJECT HIGHLIGHT:

Achieved 99% authentication reliability using Supabase Auth

Optimized UI performance by 30% with Next.js and Tailwind CSS

HOW TO RUN LOCALLY:

git clone https://github.com/mdamj/task-tracker.git

cd task-tracker

npm install

npm run dev
